Understanding AngularJS

The 3 Steps to Get Started

February 8, 2015 by Cody Alan Sanders

AngularJS, by definition, is "a structural framework for dynamic web apps." It fulfills this purpose by extending the syntax of HTML and allowing you to dynamically reference and insert data. In addition, AngularJS has its own powerful set of functions to enhance how you display your HTML elements.

While there is an extensive amount of powerful code within the Angular library, any developer with experience with HTML, CSS and Javascript can easily utilize the basics by understanding 3 main components.

3 Primary Components

Modules

Modules are contained within a Javascript file and serve to actually create the logic of the Angular portion of the project. Here the Angular app is declared, named and controllers are created.

A module may contain a number of controllers which can be called from sections of the HTML and dynamically change elements. Below you will see an initialized application with a controller.

(function() {
	var App = angular.module('App', []);

	App.controller('TutorialController', function(){
		this.links = courses;
});

Directive

A Directive is a marker on a HTML tag that tells Angular to run or reference some JavaScript code

Directives can create a reference to an app, a controller or call a function.

body ng-controller="TutorialController as tutorial"

Expression

Expressions allow you to insert dynamic values into your HTML.

They are utilized by placing double curly braces where you need data and using proper controller aliases and data names.
